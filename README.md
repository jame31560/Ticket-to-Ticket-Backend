# Ticket-to-Ticket-Backend

## Install package

Note: Recommend using virtual environment.

```shell
(ENV)$ pip3 install -r requirement.txt
```

## Create Config File

Copy `config.py.sample` as `config.py` and change value in it.

```shell
(ENV)$ cp config.py.sample config.py
```

## Run app (Local)

```shell
(ENV)$ python api.py
```
